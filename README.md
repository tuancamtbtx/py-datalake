# PyTrust
A shared python package support Data Engineering Task
## Run Test Locally
**Must Have**
```
export PYTHONPATH=$PYTHONPATH:/pydata
```
**Run Test**
```
make test
```
## Build Lib
```
make dist
```
## Usage This Lib
**Copy pydata-1.0.0.tar.gz in dist folder to your project**

**Configure requirements.txt**
```
./packages/pydata-1.0.0.tar.gz
```
